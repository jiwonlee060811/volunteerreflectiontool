# Sightshare Volunteer Reflection Survey Tool
An AI-powered anonymous survey tool for collecting and analyzing volunteer reflections and experiences.

## Files
- `index.html` - Main HTML structure
- `styles.css` - Styling and design
- `script.js` - Functionality and AI logic

## How to Use
1. Open `index.html` in a web browser
2. Fill out the volunteer reflection survey
3. Submit the form to receive AI-powered suggestions
4. Use the dashboard to view all stored reflections
5. Use the chat feature for any questions
